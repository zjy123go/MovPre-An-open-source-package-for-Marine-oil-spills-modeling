MovPre：An open-source package for Marine oil spills modeling
MovPre：An open-source package for Marine oil spills modeling is Python library which offers a simulation of oill spills.


1. Requirements Windows 10 operating system

Python 3.6
anaconda
spyder3.2.8
matplotlib (installed automatically)
numpy (installed automatically)
2. Installation
just need to put the MovPre with your main program ,then you can call it .do not need to Install it.
